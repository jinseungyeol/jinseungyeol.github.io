사업분석 보고서

사업분석 보고서는 매월 당월까지의 매출 · 매입 · 인건비 데이터를 수집하여
누적 손익 현황을 분석하고 사업 현황을 점검해 볼 수 있습니다.
(홈택스 신고 데이터 기준)

  • 총 매출

  • 전자 세금계산서 / 전자 계산서 / 현금 영수증 / 카드 · 제로페이 / 온라인(PG · 배달 · 쇼핑) /
    영세율 수출실적명세서 / 수기입력

  • 총 매입

  • 전자 세금계산서 / 전자 계산서 / 현금 영수증 / 카드 / 수기 입력

  • 인건비

    원천징수이행상황신고서

  • 근로 · 일용소득 / 사업소득 / 기타소득 / 퇴직소득

  • 예상 손익

    매출 - ( 매입 + 원천징수 )

매출 보고서

매출 보고서는 월 · 분기 · 반기 · 연간 매출 데이터를 비교할 수 있는 표와 함꼐
전년과 올해 데이터를 분석하여 추세를 볼 수 있는 차트를 제공해 드립니다.
(홈택스 신고 데이터 기준)

  • 매출 집계

    월 / 분기 / 반기 / 연간

  • 전자세금계산서 / 전자계산서 / 현금영수증 / 카드 · 제로페이 / 온라인(PG · 배달 · 쇼핑)
    영세율 수출실적명세서 / 수기 입력

  • 매출 추이

  • 매출 구성비율, 연간 매출 현황 , 거래처 매출 Top5

전년과 올해 데이터를 분석하여 추세를 볼 수 있는 차트를 제공해 드립니다.

  • 매입 구성비율, 연간 매입 현황 , 거래처 매입 Top5

예상 부가세 보고서

예상 부가세 보고서는 매출 · 매입 · 공제 세액과 예정고지세액을 수집하여
부가가치 비율과 함께 예상 부가세 납부(환급)세액을 미리 안내해 드립니다.

  • 매출 세액

  • 전자세금계산서 / 전자계산서 / 현금영수증 / 카드 · 제로페이 / 온라인(PG · 배달 · 쇼핑) / 수기 입력

  • 매입 세액

  • 전자세금계산서 / 전자계산서 / 현금영수증 / 카드 / 수기 입력

  • 의제매입세액 / 신용카드 발행세액

  • 예상 납부(환급) 세액

    매출 - ( 매입 + 공제 + 예정고지 )

거래처 매출장 보고서

거래처 매출장 보고서는 분기 · 반기 전자(세금)계산서의 거래내역을 분석할 수 있으며,
부가세 신고 전 거래처의 거래내역을 종합적으로 검토할 수 있습니다.
(홈택스 신고 데이터 기준)

  • 거래처 매출장 집계

    분기 / 반기

  • 전자 세금계산서 / 전자 계산서

  • 거래처 매출장 추이

  • 매출장 거래처 목록(상세), 직전기 비교 표

인건비 보고서

인건비 보고서는 위멤버스 급여관리에 등록된 급여대장 기준으로 반영되며
월급여 현황을 비교하고, 매출 · 매입 대비 인건비 구성비율을 분석해 드립니다.
(생성기준 : 1순위 - 위멤버스 > 급여관리 앱 > 급여대장, 2순위- 세무사랑 > 급여전송)

세무사랑 급여전송 기능이 궁금해요

  • 월 급여 대상자

  • 직원 별 세전 · 세후 급여내역 상세

  • 인건비 추이

  • 매출 / 매입 대비 인건비 구성비율, 연간 인건비 현황

부가세 신고분석 보고서

부가세 신고분석 보고서는 부가세 신고납부 기간인 1 · 4 · 7 · 10월
실시간 정보수집을 통해 데이터가 수집되며, 납부서와 함께 부가세 추이를 분석해 드립니다.
(홈텍스 신고 데이터 기준)

  • 부가가치 산출

    ((매출 - 매입) / 매출) x 100

  • 부가율 / 카드 · 현금영수증 / 의제매입금액

  • 비율 현황

  • 부가세 신고 현황 비교 추이

  • 확정 납부(환급) 세액

  • 부가가치세 접수증 / 신고서 / 매월 부가세 사전 준비금 안내
