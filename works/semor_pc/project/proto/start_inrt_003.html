홈택스를 연결해볼까요?

매월 보고서 데이터 자동수집 설정

여기저기 분산된 홈택스 수임처 정보, 일일이 파악하느라 어려우셨나요?
자동수집용 인증서 등록하고 매월 사업분석 · 매출 · 매입 보고서 데이터 알아서 척척 수집해 드려요.

1.자동수집용 홈택스 인증서 등록

[자동수집용 인증서 등록방법]

등록한 부서사용자 계정은 홈택스 자료 자동수집 실행 시 연동되어요.

자동수집용 인증서가 등록되었습니다.
2.자동수집용 홈택스 인증서 등록

등록한 부서사용자 계정은 홈택스 자료 자동수집 실행 시 연동되어요.

주의

1. 홈택스 ID/PW가 아닌 세무대리인 부서사용자 ID/PW로 등록해 주세요.

2. 홈택스 마스터 계정은 등록하지마세요. (등록 시 홈택스 이중로그인으로 접속 충돌이 발생됩니다.)

3. 위멤버스클럽 수임처앱에 등록된 부서사용자 ID는 자동수집용으로 활용됩니다. 자주 사용하지 않는 부서사용자 ID로 등록 권장드립니다.

4. 이미 등록한 부서사용자 ID는 변경 시, 익일 반영되므로 신중히 등록해 주세요. (홈택스 이중로그인으로 인한 오류 문의가 많이 접수되고 있습니다!)

홈택스 실시간 정보수집 설정

실시간 정보수집용 인증서 등록하면 홈택스를 로그인하지 않아도 거래처 별 홈택스 기본정보와
전년도 1월부터 당월까지의 정보(매출, 매입, 부가세, 원천세)를 수집해 보고서에 업데이트해 드려요.

[실시간 정보수집 인증서 확인방법]

세모리포트 > 실시간정보수집 > 인증서 설정

매월 보고서 데이터 자동수집 설정
홈택스 실시간 정보수집 설정
세무대리인 관리번호 등록

홈택스 실시간 정보수집 부서사용자 설정

홈택스 자동 로그아웃 방지를 위해 부서사용자 계정을 추가로 연동하는 것을 권장해요.

주의

1. 홈택스 ID/PW가 아닌 세무대리인 부서사용자 ID/PW로 등록해 주세요.

2. 홈택스 마스터 계정은 등록하지마세요. (등록 시 홈택스 이중로그인으로 접속 충돌이 발생됩니다.)

부서사용자 계정이 없다면? 부서사용자 계정을 잊어버렸어요!
