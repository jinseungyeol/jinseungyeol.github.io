# 디즈니플러스 클론 웹앱

> 사이드 프로젝트 · 2개월 · React, Styled Components, Firebase, Axios

TMDB API를 활용해 인기 콘텐츠를 실시간으로 제공하는 OTT 플랫폼 클론. Firebase 인증으로 로그인·회원가입을 구현하고, 반응형 UI로 다양한 디바이스에서 동일한 경험을 제공합니다.
