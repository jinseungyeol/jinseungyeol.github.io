# 링크패스

> 퍼블리싱 · 2024.06 ~ 2024.12 · 참여 100% · HTML, CSS, jQuery, Chart.js

세무 컨설팅을 위한 **자료수집 현황을 시각적으로 제공하는 웹 앱** '링크패스'의 퍼블리싱을 전담한 프로젝트입니다. 복잡한 수치 데이터를 Chart.js 그래프와 w2ui 그리드로 직관적으로 표현하고, 컨설팅·자료수집·CRM·랜딩 등 서비스 전 영역의 PC 웹 화면을 구축했습니다.

## 프로젝트 배경

사용자가 여러 지역과 보상 내역을 한눈에 비교할 수 있도록 돕는 것이 서비스의 목표였습니다. 표와 숫자만으로는 전달이 어려운 데이터를 **그래프와 정렬·필터 가능한 그리드로 시각화**하는 것이 핵심 과제였고, 퍼블리싱 단계에서 시각화 레이아웃과 인터랙션 설계까지 함께 진행했습니다.

## 주요 작업

| 영역 | 내용 |
|---|---|
| UI 퍼블리싱 | HTML/CSS로 구조를 세팅하고 jQuery로 인터랙션 구현 |
| 데이터 시각화 | Chart.js로 현황 데이터를 라인+바 믹스 차트, 도넛 차트 등으로 시각화 |
| 그리드 UI | w2ui Grid로 다중 정렬·필터가 가능한 표 기반 데이터 UI 구축 |
| 반응형 최적화 | CSS 미디어 쿼리로 노트북 해상도까지 가독성·가용성 검증 및 보완 |
| 크로스 브라우징 | 브라우저·디바이스별 동작 테스트 후 최종 UI 마무리 |

## 구현 포인트

- **Chart.js 시각화** — 랜딩·현황 화면에서 라인과 바를 겹친 믹스 차트, 비율 표현용 도넛 차트 등을 구성해 수치 데이터를 한눈에 비교할 수 있게 했습니다.
- **팝업 시스템 정형화** — 센터 레이어 팝업(centerLayerPop), 확인 팝업(confirmPop), 토스트 팝업(toastPop)으로 팝업 유형을 규격화하고 일련번호 채번 규칙으로 관리했습니다.
- **공통 리소스 체계** — Pretendard 웹폰트, Swiper 슬라이더, Lottie 기반 로딩 애니메이션(JSON), SVG 아이콘 세트를 공통 리소스로 정리해 화면 간 일관성을 유지했습니다.
- **오류·예외 화면까지 커버** — 네트워크 오류, 세션 만료, 정기 점검, 열람 불가 등 예외 상황 화면을 별도 카테고리로 구축해 서비스 전 흐름을 빠짐없이 퍼블리싱했습니다.

## 데모 구성

- 카테고리 탭(공통·컨설팅·자료수집·고객 CRM·랜딩·환경설정·팝업·종료 등)으로 화면 산출물을 분류한 목록 페이지에서 각 화면을 열어볼 수 있습니다.
- 컨설팅 자료수집 단계별 화면, Chart.js 차트가 들어간 랜딩 화면, w2ui 그리드 화면, 각종 팝업·예외 화면이 포함되어 있습니다.

## 성과

- **Chart.js를 활용한 데이터 시각화**로 복잡한 세무 데이터를 직관적인 화면으로 전달
- jQuery + Chart.js + w2ui 조합으로 동적 UI와 표 기반 데이터 표현을 동시에 구현하는 경험 축적
