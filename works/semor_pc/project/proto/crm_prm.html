#01 ‘부가세 절세진단' 제공

‘부가세 절세진단' 서비스는 세무사무소 고객이 접속해 간편인증 후
고객의 상세 정보를 수집하여예상되는 절세금액을 계산해주는 서비스입니다.

1단계

부가세 절세진단
소개 페이지

차별화된 부가세 절세 진단 서비스를 소개하는 화면으로
세무사무소의 로고 등록과 카카오 채널 연동을 지원하며,
마케팅 유입 고객에게 자동 알림을 제공합니다.

2단계

고객 데이터 수집

무료 부가세 절세 진단 버튼을 클릭하면, 1분 간편 인증과
함께 홈택스에서 고객 정보와 절세 분석을 자동으로 수집
하여 상담에 활용합니다.

3단계

진단 결과
및 안내

카카오톡 또는 KT PASS 간편 인증 후, 부가세 절세진단
결과 페이지에서 고객 유형에 따라 예상 절세금액 안내
화면을 제공합니다.

4단계

부가세 신고분석
보고서 발송

지난 세금과 비교 추이를 제공하며, 앞으로 매월 예상 세금을
파악하여 고객이 원하는 정보를 미리 알려줄 수 있는
부가세 신고 분석 보고서를 제공합니다.

부가세 절세진단 샘플

#02 ‘브릿지 페이지' 제공

‘브릿지 페이지' 서비스는 세무사무소 고객의 정보를 수집하고
샘플 보고서를 발송하여 차별화된 세무서비스를 홍보할 수 있습니다.

1단계

부가세 절세진단
소개 페이지

프리미엄 세무 서비스를 신청할 수 있는 사장님을 위한
페이지로, 차별화된 솔루션 서비스를 소개하는
브릿지 페이지를 제공합니다.

2단계

고객 데이터 수집

사업경영과 프리미엄 세무 보고서를 체험해 볼 수 있는
상담을 신청하시면, 고객 정보를 수집하여 혜택을
드릴 수 있도록 맞춤형 서비스를 지원합니다.

3단계

진단 결과
및 안내

세무사무소 카카오채널이 연결된 경우 해당 채널로
샘플보고서를 발송하고, 연결 되지 않은 경우에는
세모리포트 채널로 발송해드립니다.

궁금증을 해결해드려요!

  • 대표님들에게 상담신청이 가능한가요?

    네, 가능합니다!
    세무사무소 홈페이지에서 해당 페이지 제공 후, 대표님들의 정보를 미리알고 상담 할 수 있습니다.

  • 고객의 어떤 정보를 미리 알 수 있나요?

    고객정보, 사업자정보, 예상절세금액, 매출금액, 납부세액 등을 미리 확인 가능합니다.

  • 서비스 요금은 어떻게 되나요?

    해당 서비스는 세모리포트의 차별화 된 부가서비스로, 구간 요금제 이상 사용 고객은 무료로 제공됩니다.

  • 부가세 절세 진단을 돌리면 세무사무소에는 어떻게 전달 되나요?

    먼저, 어드민을 통해 접수 내역 확인 가능합니다. ( 세모리포트 > 고객관리 > 세모CRM )

    또한, FLOW로 실시간 접수 내역을 알려드립니다. ( 플로우 : https://flow.team/invitation/RAzu_NHGp )

  • 법인사업자도 가능한가요?

    개인사업자는 물론, 법인사업자도 모두 가능합니다.

  • 그래서 “부가세 절세 진단"을 실행하면 어떻게 나올까요?

    나의 고객 : 고객에게 “훌륭하게 신고되었습니다.” 라고 안내 (세모리포트 “고객관리”에 등록고객 전체 )
    다른 세무사의 고객 : “접수 되었습니다.” 로 구분되어 나옵니다.

  • “절세가능 금액”은 어떤 의미인가요?

    · 절세금액 = 세모 경비율(업종별)에 따른 세금 - 납부세금

    · 세모경비율 : 표준 경비율 + 세모고객통계 반영

    · 마이너스(-) : 납부세금이 부가율 대비 많고, 절세 가능 금액이 없을 확율이 높다는 것을 말합니다.

    · 플러스(+) : 절세가능 금액이 있다는 것을 말합니다.

  • FLOW는 유료로 사용해야 하나요?

    게스트로 가입하셔서 사용하시면 무료로 조회 사용 가능합니다.
