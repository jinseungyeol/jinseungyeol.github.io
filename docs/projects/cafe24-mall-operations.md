# 카페24 자사몰 통합 운영

> 개발 · 참여 100% · HTML, CSS, JavaScript, GSAP, GitHub Actions, Figma, Tokens Studio

웰니스 D2C 13개 브랜드의 카페24 자사몰 운영 체계. **퍼블리셔 3인이 4개 브랜드씩 분담하던 운영을 Claude Code 기반 작업 파이프라인을 직접 설계해 1인 전담으로 전환**했습니다.

## 핵심 내용

- **스킨 코드의 git 이관** — admin 에디터·FTP로 직접 고치던 스킨을 저장소로 옮겨 이력·리뷰가 남게 하고, push하면 변경분만 카페24 스킨에 자동 배포
- **카페24 특유의 함정 해결** — 일반 배포 액션과 비호환인 FTP를 lftp로 전환, 대소문자만 다른 중복 파일(phantom)을 배포 전 게이트로 차단, 봇 커밋이 후속 워크플로우를 트리거하지 않는 GitHub 정책을 직접 호출로 우회, 전체 덮어쓰기 대신 증분 배포
- **AI 보조 워크플로우** — 카페24 스킨 구조를 이해시킨 Claude Code로 자연어 요청 → 작업 → 배포 흐름을 구축하고, 반복 인터랙션(옵션 레이어, 다중 캐러셀, 프로모션 로직)을 공통 패턴화
- **디자인 기준의 도구화** — 실제 운영 페이지를 Figma로 그대로 추출하는 플러그인을 직접 구현해 레퍼런스를 확보하고, 색상·타이포그래피·간격·모서리를 디자인 토큰으로 표준화해 Tokens Studio와 코드의 기준을 일치. atoms→organisms를 실사용 화면 기준으로 도출
