# 로움아이티

> 퍼블리싱 · 2022.06 ~ 2022.12 · 참여 100% · HTML, CSS, jQuery, AWS(S3, CloudFront, Route 53)

로움아이티의 핵심 서비스를 소개하고 브랜드 신뢰도를 높이기 위한 **기업 공식 홈페이지 구축** 프로젝트입니다. 기획 일부 참여부터 퍼블리싱, 프론트 개발, AWS 기반 배포까지 전 과정을 참여율 100%로 단독 수행했습니다. 방문자가 기업의 주요 제품과 비전을 직관적으로 이해할 수 있도록 **스크롤 인터랙션 기반의 콘텐츠 흐름**을 설계하고, **카카오 지도 API로 위치 정보**를 시각적으로 제공하는 것이 주요 목표였습니다.

## 프로젝트 배경

- 회사의 서비스·비전을 소개하고 브랜드 신뢰도를 높이기 위한 공식 홈페이지가 필요했습니다.
- 단순 정보 나열이 아니라, 스크롤을 따라 브랜드 메시지가 자연스럽게 전달되는 몰입형 구성을 지향했습니다.
- 기획 단계에 일부 참여하여 기업 소개 목적에 부합하는 정보 구조와 콘텐츠 흐름을 함께 논의했습니다.

## 주요 작업

| 영역 | 내용 |
|---|---|
| 기획·디자인 협의 | 브랜드 메시지와 콘텐츠 흐름에 맞춘 와이어프레임 및 시안 검토 |
| 퍼블리싱 | 시맨틱 마크업과 접근성을 고려한 HTML/CSS 구조화 |
| 스크립트 개발 | 순수 바닐라 자바스크립트로 스크롤 트리거 애니메이션·요소 전환 효과 구현 |
| 지도 연동 | 카카오 지도 API로 오피스 위치 및 마커 정보 노출 |
| 반응형 대응 | 뷰포트별 콘텐츠 재배치, 폰트·간격 조절, 모바일 최적화 |
| 배포·운영 | AWS S3 정적 호스팅 + CloudFront CDN + Route 53 도메인 설정, 캐시 관리 |

## 구현 포인트

- **바닐라 JS 스크롤 인터랙션** — 별도 라이브러리 없이 스크롤 이벤트 기반으로 섹션 진입 시 애니메이션과 요소 전환을 처리해, 가볍게 동작하면서도 사용자의 몰입을 유도했습니다.
- **카카오 지도 API 연동** — 오시는 길 섹션에 지도를 삽입하고 마커·오피스 정보를 표기해 위치 정보를 텍스트가 아닌 시각 정보로 제공했습니다.
- **AWS 정적 배포 파이프라인** — S3에 정적 산출물을 배포하고 CloudFront로 전파 속도를 개선했으며, 배포 시 캐시 무효화까지 직접 관리했습니다. Route 53으로 도메인을 연결해 라이브 서비스로 운영 중입니다.

## 배운 점

- 단순한 소개 페이지라도 브랜드 메시지를 전달하려면 구조화된 콘텐츠 설계가 중요하다는 점을 체감했습니다.
- 스크롤 인터랙션은 과하지 않게 쓸 때 사용자의 몰입을 유도하는 효과적인 수단임을 확인했습니다.
- 퍼블리싱을 넘어 배포·도메인·CDN까지 다루며 정적 사이트 운영의 전체 흐름을 경험했습니다.
