카드매출 입금보류란?


- 취소매출 등의 원인으로 카드사로부터 입금이 보류된 내역입니다.


- 계속 입금보류 상태인 경우에는 해당 카드사에 직접 확인하셔야 합니다.


- 실입금액이 보류금액보다 큰 경우 : 입금보류 기간이 종료된 후 해당 카드사에서 보류된 금액의 전체 또는 일부를 다시 입금하는 경우


- 실입금액이 마이너스 금액인 경우 : 카드사로부터 입금까지 완료된 건이 입금 이후에 취소가 등록되어 카드사에서 이미 입금된 대금을 다시 회수하기 위해 이전 또는 이후에 발생하는 해당 카드사의 매출을 보류하는 경우


- 확인필요 로 보이는 데이터는 입금보류 상태로 장기간 남아있는 건으로 해당 카드사에서 직접 확인하셔야 합니다.
