주휴수당이란?

  • 일주일에 15시간 이상 일한 근로자에게 일주일에 하루씩 유급휴일을 주는 제도를 말합니다.

  • 다만, 다음 주휴수당 조건에 충족하지 않으면
    예외대상이 됩니다.

[예외대상]

  1. 1. 일을 한 주에 하루 이상 결근이 있을 경우
  2. 2. 근로 기간이 일주일 미만인 경우
  3. 3. 4주 평균 주 15시간 미만인 경우

  • 통상적으로 40시간 미만 근로자의 주휴수당은 다음과 같이 계산합니다.

  • 근로시간(소정) / 40시간(법정 1주 근로시간) x
    8시간(법정 일일 근로시간) 시급
    = 일주일 소정근로시간 / 5 x 시급

  • 예시) 월, 수, 금에 6시간씩 근무하는 직원의 경우,

    • 시급 : 10,000원

    • 1일 소정근로시간 : 6시간

    • 주 소정근로일 : 3일

    • 주휴수당 :
      6 (1일 소정근로시간) x 3 (주 소정근로일)
      / 5 x 10,000 (시급) = 36,000원
